# Security Operations Guide — Microsoft 365 Defender XDR

Security watchdog code and operational guides organized by Microsoft Defender product areas for threat detection, investigation, and response.

## Overview

This repository contains security operations code and documentation for Microsoft 365 Defender (XDR) components. Materials are organized by product area with guidance for threat hunters, security analysts, and operations teams.

## Contents

### Security Product Areas

- **EntraID/** — Microsoft Entra ID (formerly Azure AD) security guidance, identity-based threat detection, and authentication abuse scenarios
- **MDA/** — Microsoft Defender for Apps security controls, cloud app discovery, and SaaS security monitoring
- **MDE/** — Microsoft Defender for Endpoint threat detection, endpoint hardening, and incident response procedures
- **MDI/** — Microsoft Defender for Identity threat detection, lateral movement analysis, and identity-based attack prevention
- **MDO/** — Microsoft Defender for Office 365 email security, phishing prevention, and mailbox threat protection
- **XDR/** — Cross-platform threat response and correlation across all Microsoft Defender products

### Documentation

- **Requirements.md** — Prerequisites and requirements for deployment and operations
